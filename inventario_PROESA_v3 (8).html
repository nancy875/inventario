🔄 Sincronización en tiempo real (Firebase)
¿Qué hace esto? Cuando Emilio cambia algo, Nancy lo ve actualizado al instante — y viceversa. Todos los usuarios del equipo trabajan sobre los mismos datos en tiempo real.
Costo: Gratis (Firebase Spark plan — 1GB almacenamiento, 10GB transferencia/mes, más que suficiente).
Pasos para activar (5 minutos):
  1. Ve a console.firebase.google.com → crea un proyecto (ej: "proesa-inventario")
  2. En el proyecto → Realtime Database → Create Database → elige modo de prueba → selecciona la región más cercana (us-central1)
  3. Ve a Project Settings (ícono ⚙) → General → baja hasta "Your apps" → clic en </> (Web) → registra la app
  4. Copia el objeto firebaseConfig que aparece y pega cada campo abajo
  5. En Realtime Database → Rules cambia las reglas a: ".read": true, ".write": true y pública

⚙ Configuración de envío automático de correos
Cómo obtener tu API Key de Brevo (gratis):
  1. Entra a app.brevo.com → crea cuenta gratis
  2. Ve a My Account → SMTP & API → API Keys
  3. Clic en "Generate a new API key" → copia la clave
  4. Pégala abajo junto con el correo remitente (debe estar verificado en Brevo)
Cómo obtener tu API Key de Resend (gratis):
  1. Entra a resend.com → crea cuenta gratis
  2. Ve a API Keys → clic en "Create API Key"
  3. Copia la clave y pégala abajo
  4. El correo remitente debe ser onboarding@resend.dev hasta que verifiques tu dominio
Cómo obtener tu API Key de SendGrid (gratis):
  1. Entra a app.sendgrid.com → crea cuenta gratis
  2. Ve a Settings → API Keys → Create API Key
  3. Selecciona "Full Access" → copia la clave
  4. Verifica tu correo remitente en Settings → Sender Authentication
Debe estar verificado en el servicio elegido
